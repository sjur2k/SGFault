sgfault - The SGFault Compiler
===============================

USAGE:
    sgfault <file.sg> [flags]

ARGUMENTS:
    <file.sg>          Path to source code. Must have .sg file extension.

FLAGS:
    -o, --output       Next argument sets custom binary output name.

    -d, --output-dir   Next argument sets custom binary output directory.

    -v, --verbose      Enable verbose output. Ommits deletion of intermediary assembly and object files.

    -h, --help         Show this help message and exit with code 0.

EXAMPLES:
    sgfault myFile.sg
    sgfault myFile.sg -o myProgram
    sgfault myFile.sg --verbose
    sgfault src/myFile.sg -o build/myProgram -v

OUTPUT:
    Default output name is the same as input source file name.
    Default output directory is the current directory.

    - File names must be 100 characters or fewer.
    - Flags can be in any order after the source file.
