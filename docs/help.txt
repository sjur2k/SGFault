sgfault - The SGFault Compiler
===============================

USAGE:
    sgfault [options] <source_file>

REQUIRED:
    <source_file>           Absolute or relative path to source code file. Must have .sg file extension.

OPTIONS:
    -o, --output <path>     Sets custom output path. Accepts both relative and absolute paths.

    -v, --verbose           Enable verbose output. Keeps intermediary assembly and object files.

    -h, --help              Show this help message and exit with code 0.

EXAMPLES:
    sgfault my_file.sg
    sgfault -o my_program my_file.sg 
    sgfault my_file.sg --verbose
    sgfault -o ~/my_project/build/my_program -v src/my_file.sg

OUTPUT:
    Default output path is the same as input.
    sgfault path/name.sg -> path/name

NOTES:
    - File paths must not be longer than 4096 characters.
    - Options can be in any order between compiler command name and the source file.
